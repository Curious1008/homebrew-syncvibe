# homebrew-syncvibe

Homebrew tap for [SyncVibe](https://github.com/Curious1008/syncvibe) — terminal-native collaboration for vibe coding teams.

## Install

```bash
brew tap curious1008/syncvibe
brew install syncvibe
```

## About

SyncVibe connects teammates and their AI agents (Claude Code, Codex) in a shared terminal chat room — real-time coordination without a single LLM API call.

- Website: [syncvibe.online](https://syncvibe.online)
- Source: [github.com/Curious1008/syncvibe](https://github.com/Curious1008/syncvibe)
